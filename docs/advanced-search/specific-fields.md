# الحقول المحددة

يتيح البحث المتقدم إمكانية تحديد **حقل محدد** للبحث فيه، وهو ما يوفر دقة أكبر في النتائج مقارنة بالبحث في جميع الحقول.

## الحقول المتاحة

| الحقل | الوصف |
|-------|-------|
| **جميع الحقول** | البحث عن كلمات البحث في جميع الحقول دون تحديد حقل معين |
| **عنوان المقال أو الرسالة** | قصر البحث على الكلمات الواردة في العنوان فقط |
| **عنوان الدورية أو المؤتمر** | البحث ضمن عنوان المجلة أو المؤتمر |
| **المؤلف** | البحث عن المؤلف أو المترجم أو المحقق |
| **الموضوع** | البحث عن المصطلحات الموضوعية المضافة من قبل المفهرسين |

## مثال

للبحث عن أبحاث لمؤلف معين في مجال التربية:
1. اختر حقل **المؤلف** وأدخل اسم المؤلف.
2. أضف صف بحث آخر واختر حقل **الموضوع** وأدخل كلمة "التربية".
3. استخدم **AND** للربط بين الشرطين.
