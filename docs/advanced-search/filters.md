# الفلاتر المتنوعة

يتيح البحث المتقدم مجموعة من **الفلاتر** التي يمكن تطبيقها قبل تنفيذ عملية البحث للحصول على نتائج أكثر دقة.

## أنواع الفلاتر

### 1. نوع المحتوى

بإمكان الباحث قبل تنفيذ عملية البحث أن يحدد **نوع المادة** المراد استرجاعها:

| النوع | الوصف |
|-------|-------|
| **بحوث/مقالات** | المقالات والدراسات العلمية |
| **بحوث مؤتمرات** | أبحاث المؤتمرات والندوات |
| **رسائل جامعية** | رسائل الماجستير والدكتوراه |
| **نصوص أدبية** | النصوص والمتون الأدبية |
| **عروض كتب** | مراجعات وعروض الكتب |

### 2. سنة النشر

من خلال هذا الفلتر يستطيع الباحث تحديد **فترة زمنية محددة** للمواد التي يرغب في الحصول عليها، سواء سنة بعينها أو عدة سنوات.

هناك طريقتان لتقييد البحث بسنة النشر:

1. **إدخال سنة أو سنوات بشكل يدوي** في حقل النص.
2. **استخدام الشريط الزمني** (Timeline Slider) لتحديد نطاق زمني مرئي.

:::tip
استخدم الفلاتر قبل بدء البحث لتقليل عدد النتائج غير المرغوب فيها منذ البداية.
يمكنك أيضاً تضييق النطاق أكثر بعد عرض النتائج من خلال **[فلاتر الجانب](/docs/search-results/narrowing)**.
:::
