# نظرة عامة على البحث المتقدم

**البحث المتقدم** هو الخيار المفضل للباحث المتمرس، لما يقدمه من إمكانيات وتقنيات متقدمة تجعل الباحث قادراً على **التحكم بالبحث** وتصفية نتائجه بشكل دقيق.

## إمكانيات البحث المتقدم

يوفر البحث المتقدم مجموعة من الأدوات المتطورة:

| الإمكانية | الوصف |
|-----------|-------|
| **[الحقول المحددة](/docs/advanced-search/specific-fields)** | البحث ضمن حقل معين (جميع الحقول، العنوان، المؤلف، الموضوع، المصدر) |
| **[عوامل الربط المنطقية](/docs/advanced-search/boolean-operators)** | استخدام AND، OR، NOT للربط بين المصطلحات |
| **[البحث بالمجموعات](/docs/advanced-search/group-search)** | إنشاء مجموعات بحث متعددة لصياغات معقدة |
| **[البحث بالعبارة](/docs/advanced-search/phrase-search)** | البحث عن عبارة محددة باستخدام علامات التنصيص |
| **[فلترة النتائج](/docs/advanced-search/filters)** | تحديد نوع المحتوى وسنة النشر قبل البحث |

:::tip
استخدم البحث المتقدم عندما تحتاج إلى:
- البحث في حقول محددة
- الجمع بين مصطلحات متعددة بعلاقات منطقية
- صياغة استعلامات بحث معقدة
:::
