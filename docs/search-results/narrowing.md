# تضييق نطاق البحث

يمكن تقليل عدد النتائج عن طريق تضييق نطاق البحث باستخدام القائمة الموجودة على يسار الشاشة.

## فلاتر تضييق النطاق

| الفلتر | الوصف |
|--------|-------|
| **نوع المحتوى** | تصفية حسب نوع المادة (بحوث، رسائل جامعية، مؤتمرات، إلخ) |
| **قاعدة المعلومات** | تصفية حسب قاعدة البيانات المحددة |
| **المؤلف** | تصفية حسب اسم المؤلف |
| **سنة النشر** | تحديد نطاق زمني معين |

:::tip
استخدم هذه الفلاتر لتقليل عدد النتائج والوصول إلى المواد الأكثر صلة ببحثك.
من المستحسن استخدام فلتر واحد أولاً ثم إضافة المزيد حسب الحاجة.
:::
