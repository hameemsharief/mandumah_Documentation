# النص الكامل

يتم عرض النصوص الكاملة للبحوث والدراسات في ثلاثة أشكال مختلفة.

## صيغ العرض

| الصيغة | الوصف | نسخ ولصق | حجم الملف |
|--------|-------|-----------|-----------|
| **PDF صورة** | نسخة مصورة من الأصل المطبوع | غير متاح | كبير |
| **PDF نص** | نسخة أعيدت طباعتها كنص | متاح | صغير |
| **HTML** | نسخة على شكل صفحة ويب | متاح | صغير |

### PDF صورة
نسخة ممسوحة ضوئياً مطابقة تماماً للأصل المطبوع، ولكنها لا توفر إمكانية النسخ واللصق لنصوصها.

### PDF نص
النسخة التي أعيد إدخال نصوصها كاملة كنص ومن ثم تحويلها إلى صيغة PDF. وهي تسمح بالنسخ واللصق، إضافة إلى صغر حجم الملف.

### HTML
نص المقال كاملاً تم تحويله إلى صفحة ويب لا تحتاج إلى تحميل، بل يتم استعراضها مباشرة. تتميز هذه النسخة أيضاً بإمكانية النسخ واللصق.

:::tip
استخدم نسخة **HTML** للعرض المباشر السريع، ونسخة **PDF** للحفظ والطباعة.
:::
