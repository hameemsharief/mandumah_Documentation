# ترتيب نتائج البحث

يمكن ترتيب نتائج البحث من القائمة المنسدلة الموجودة أعلى الشاشة جهة اليسار.

## خيارات الترتيب

| الترتيب | الوصف |
|---------|-------|
| **مدى الصلة** | ترتيب حسب درجة الصلة بين كلمات البحث والوثائق المسترجعة **(الخيار الافتراضي)** |
| **توفر النص الكامل** | ترتيب حسب توفر النص الكامل للمواد المسترجعة |
| **التاريخ تنازلياً** | من الأحدث إلى الأقدم |
| **التاريخ تصاعدياً** | من الأقدم إلى الأحدث |
| **الأحدث إضافة** | حسب تاريخ الإضافة إلى قواعد المعلومات |
| **الاسم الأخير للمؤلف** | ترتيب هجائي |
| **عنوان المادة** | ترتيب هجائي |

:::tip
استخدم ترتيب **مدى الصلة** للحصول على النتائج الأكثر دقة أولاً، أو استخدم **التاريخ تنازلياً** للحصول على أحدث الأبحاث.
:::
