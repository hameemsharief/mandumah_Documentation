# تصفح الرسائل الجامعية

بالإضافة إلى **البحث المباشر** عن الرسائل الجامعية، توفر دار المنظومة طريقة أخرى للوصول إلى الرسائل وهي **التصفح المباشر**.

## كيفية الوصول

يمكن للمستفيد الضغط على الرابط الخاص بـ **تصفح جميع الرسائل الجامعية** الموجود في النصف الأسفل من الشاشة الرئيسية تحت قاعدة **المنظومة للرسائل الجامعية (Dissertations)**.

## هيكل التصفح

قسمت الرسائل الجامعية هرمياً حسب التسلسل التالي:

1. **الدول** → تصنيف أولي حسب الدولة
2. **الجامعات** → الجامعات التابعة لكل دولة
3. **الكليات** → الكليات التابعة لكل جامعة

## مثال على مسار التصفح

```
الدول
  └── المملكة العربية السعودية
        └── جامعة الملك سعود
              └── كلية التربية
                    └── عرض جميع الرسائل في هذه الكلية
```

## مقارنة بين طريقتي الوصول للرسائل

| الطريقة | الميزة |
|---------|--------|
| **البحث المباشر** | بحث سريع باستخدام الكلمات الدالة والعوامل المنطقية |
| **التصفح المباشر** | استعراض هيكلي منظم حسب الدولة ← الجامعة ← الكلية |

:::tip
استخدم **التصفح المباشر** عندما تريد استكشاف الرسائل المتاحة في جامعة أو كلية معينة، واستخدم **البحث** عندما تبحث عن موضوع محدد.
:::
