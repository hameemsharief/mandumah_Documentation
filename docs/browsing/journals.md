# تصفح الدوريات والمؤتمرات

توفر القواعد وسيلة أخرى للوصول إلى الأبحاث والدراسات غير البحث بالكلمات المفتاحية، وهي **التصفح المباشر للدوريات**. ويعني ذلك أن بإمكان الباحث اختيار مجلة علمية بعينها ومن ثم الدخول على مجلداتها وتصفح أعدادها مباشرة.

## التخصصات الرئيسية

لتسهيل التصفح، قسمت المجلات حسب التخصصات الموضوعية الكبرى لقواعد معلومات دار المنظومة:

| القسم | اسم القاعدة |
|-------|-------------|
| **العلوم التربوية والاجتماعية** | EduSearch |
| **العلوم الاقتصادية والإدارية** | EcoLink |
| **العلوم الإسلامية والقانونية** | IslamicInfo |
| **العلوم الإنسانية** | HumanIndex |
| **علوم اللغة والأدب** | AraBase |

## التخصصات الفرعية

تحت كل قسم رئيسي توجد **التخصصات الفرعية** لذلك القسم.

مثال: قسم **العلوم التربوية والاجتماعية** يندرج تحته التخصصات التالية:

- عام
- علم النفس
- علم الاجتماع والخدمة الاجتماعية
- التربية

## طريقة التصفح

1. اختر القسم الرئيسي.
2. اختر التخصص الفرعي.
3. سيعرض النظام قائمة بجميع المجلات والمؤتمرات في ذلك التخصص.
4. اختر أي مجلة أو مؤتمر.
5. سيعرض النظام جميع الأبحاث التي نشرت فيها مرتبة حسب **المجلدات** (السنة).
6. تحت كل مجلد جميع **الأعداد** التي صدرت ضمن ذلك المجلد.
7. عند اختيار أي عدد، سيعرض النظام جميع البحوث والدراسات التي يحتويها ذلك العدد.

:::tip
تصفح المجلات يشبه تجربة المكتبة الفعلية، حيث يمكنك استعراض المجلدات والأعداد كما لو كنت تتصفح الرفوف مباشرة.
:::
